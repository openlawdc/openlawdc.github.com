Our own lightweight, static hosted, fully accessible browser for the DC Code.

/home-rule-act

An open-source, simple reader for the District of Columbia Home Rule Act

archive

Our /browser was a prototype, fast, open-source browser for the 2012 unofficial DC Code. The repository /dc-decoded holds experimental parsers for the unofficial released code into content for /browser and State Decoded
